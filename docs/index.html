Why Vortex

Built Different

Every design decision optimizes for speed, privacy, and developer experience. No compromises.

Native Performance

Compiled to native machine code with Rust. No Electron, no V8, no overhead. Just raw speed on every platform.

Privacy First

Zero telemetry. Zero cloud sync. Your API keys, tokens, and data never leave your machine. Period.

Git-Friendly

Collections, environments, and tests stored as clean JSON. Version control your API workspace like code.

100% Offline

No login required. No internet needed. Works anywhere — air-gapped environments, VPNs, airplanes.

Open Source

MIT licensed. Inspect every line. Contribute, fork, customize. No vendor lock-in, no surprise paywalls.

Clean Architecture

Hexagonal architecture with strict layer separation. Domain has zero I/O. Infrastructure is swappable.

Features

Everything You Need

Full-featured API development toolkit. No bloat, just the tools that matter.

HTTP/1.1 & HTTP/2

Full protocol support via reqwest + rustls. All methods, custom headers, query params, and configurable timeouts.

Authentication

Basic, Bearer, API Key (header/query), and full OAuth 2.0 with automatic token refresh.

Collections & Folders

Organize requests into nested folders. Drag-and-drop support. Git-friendly JSON format.

Code Generation

Export requests to 13 languages: cURL, Python, JS, Rust, Go, Java, C#, PHP, Ruby, Swift, Kotlin, Dart, PowerShell.

Environments & Variables

Variable substitution with {{var}} syntax. Secret separation. Switch contexts instantly.

Testing & Assertions

Assertion-based test suites. Check status codes, headers, body content, and JSON paths.

Import & Export

Import Postman collections seamlessly. Export to HAR and OpenAPI 3.0 formats.

Scripting & WebSocket

Pre-request and post-response scripts. WebSocket connections. Cookie management and request history.
