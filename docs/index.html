Features

Everything you need for API development, with none of the bloat.

HTTP Client

HTTP/1.1 and HTTP/2 via reqwest + rustls. All methods, custom headers, query params.

Authentication

Basic, Bearer, API Key (header/query), OAuth 2.0 with token refresh.

Environments

Variable substitution with secret separation. Switch contexts instantly.

Collections

Organize requests into folders. JSON-based, git-friendly format.

Import / Export

Import from Postman. Export to HAR and OpenAPI 3.0 formats.

Code Generation

Export requests to 13 languages: cURL, Python, JS, Rust, Go, Java, C#, PHP, Ruby, Swift, Kotlin, Dart, PowerShell.

Testing

Assertion-based test suites. Status, header, body, and JSON path checks.

Scripting

Pre-request and post-response scripts for dynamic workflows.

TLS / Proxy

Client certificates, custom CA, TLS version pinning. HTTP/HTTPS/SOCKS5 proxy.

History

Automatic request history with search, filter, and replay.
